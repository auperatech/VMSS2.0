aup::avaf::VideoStreamInfoPacket
VideoStreamInfoPacket, is designed to encapsulate information about a video stream, inheriting from PacketBase and implementing two interfaces for communication: packet::IHostToDevicePacketComm and packet::IDeviceToHostPacketComm. The class serves as a data structure for holding video stream metadata and potentially facilitating communication about video stream configurations between a host and a device.
