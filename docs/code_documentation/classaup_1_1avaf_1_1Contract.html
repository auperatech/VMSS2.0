Detailed Description

Contract is the class that manages the configuration of the input and output streams of a node and calculator. Contract should be filled by defining the fill_contract() function in a calculator and passing sample packets to each sample packet list, and by setting the input_attrs_arr entry for any streams containing side packets.

An example of this can be found in box_visualizer
