class  aup::avaf::Node
 Node is a class that represents each node in the graph. It contains a calculator and manages input and output streams to that calculator. Importantly, the Node thread periodically makes a call to the execute function of a calculator, in which the calculator may call get_packet to obtain a packet from an input stream to the node for internal use. A calculator may also call Node::enqueue() to enqueue a packet to one of the output streams of the node. More...
