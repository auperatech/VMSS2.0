Detailed Description

template<typename OptionsType>
class aup::avaf::CalculatorBase< OptionsType >

CalculatorBase is the base class for defining a calculator, which is responsible for the main processing functionality of a node in a graph. Any inheritors must override the initialize() function, and should override the fill_contract() function. The execute() function may be overriden and will be called periodically by the node. It is generally used to periodically get packets from the input streams to the node.

For an example of filling these functions, take a look at box_visualizer
